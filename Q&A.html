Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML : provides access to the inner html of content an element
innerText : provides access to the inside visible content of an html element
textContent:prodives access to both visible and hidden content of a HTML element
Question-3: What is event delegation in the DOM?
This is a JS technique where, instead to adding event listeners on multiple child elements, we directly add a event listener to the parent element. When a event occurs at child,it can be handled by the parent element.this is because of the concept of event bubbling.
Question-4: What is event bubbling in the DOM?
It is a process in js, when an child element get triggered for an event it first runs of the targeted element and then moves up to its parent element until it reaches the root.
